# Contributing to CDP/VIP Protocol

Welcome to the Cognitive Sovereignty Project! We are building the foundational layer for human-AI symbiosis.

## How to Contribute
1. **CIP Proposals**: If you have an idea for improving the protocol, please submit a draft in the `/CIPs` folder following the `CIP-template.md`.
2. **Discussions**: Open an Issue to discuss philosophy, ethics, or technical implementation.
3. **Code**: Help us build the first cognitive data extraction tools.
