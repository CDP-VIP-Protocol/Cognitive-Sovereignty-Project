## ğŸ— Architecture 

The protocol is built on three minimalist, high-performance layers:


1.  **CDP (Cognitive Data Protocol)**: Standardizes how raw cognitive signals (logic, focus, emotion) are captured and hardware-attested via TEE.
    
2.  **VRP (Value Refinery Protocol)**: A community-driven layer for structuring and labeling raw data into AI-ready training packets.
   
3.  **VIP (Value Interchange Protocol)**: An algorithmic settlement engine for streaming, intermediary-free wealth redistribution.
